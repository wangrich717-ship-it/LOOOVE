# LOOOVE · 循证 AI 身心教练 Demo

面向备孕到产后女性的智能戒指身心陪伴 App 的交互式前端 Demo。

一枚智能戒指持续监测睡眠、HRV、经期等生理数据，配合 App 提供「AI + 身体照顾 + 心理调适」的一体化陪伴：

- **早安时光** — 睡眠 & 身心数据 → 抽正念卡 + 写给自己 → 个性化身心建议
- **晚安时光** — 身体回顾 → 感恩日记 → 基于当天 HRV 生成的专属冥想音乐 → 呼吸入睡
- **修心练习** — 融合 CBT、ACT、DBT、心理动力、积极心理学、正念、叙事等多流派的循证练习
- **回忆漫游** — 把每一天的身心记录，封存成妈妈的专属回忆录

## 预览

直接在浏览器打开 [`LOOOVE-demo.html`](LOOOVE-demo.html) 即可（单文件，纯原生 HTML/CSS/JS，无需构建）。
